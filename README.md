# Event-Feed

Проект "Event-Feed" — минимальное React-приложение, собранное с помощью Vite. Это учебный/демонстрационный проект для отображения ленты событий и практики работы с современным стеком фронтенда.

**Коротко:** отображение списка событий, быстрый перезапуск в режиме разработки и простая структура проекта, готовая к расширению.

**Технологии:**
- React 19
- Vite (с HMR)
- ESLint для базовой проверки кода

**Требования**
- Node.js 18+ (рекомендуется LTS)
- npm или yarn

**Установка и запуск**
1. Открой терминал и перейди в папку проекта:

```
cd Event-Feed
```

2. Установи зависимости:

```
npm install
```

3. Запуск в режиме разработки (с HMR):

```
npm run dev
```

4. Сборка для продакшна:

```
npm run build
```

5. Локальный предпросмотр собранной версии:

```
npm run preview
```

6. Запуск линтинга:

```
npm run lint
```

**Структура проекта (основное)**
- `Event-Feed/` — корневая папка фронтенд-приложения
  - `index.html` — HTML-шаблон
  - `src/` — исходники приложения
    - `main.jsx` — точка входа
    - `App.jsx` — главный компонент
    - `App.css`, `index.css` — стили
  - `package.json` — npm-скрипты и зависимости
  - `vite.config.js` — конфигурация Vite
